103

::: details

“简单”的反义词是复杂，而非“杂乱”，排除；

104

B 胚胎∶婴儿  
C 种子∶花朵  
D 蝌蚪∶青蛙  
::: details

“丑小鸭”和“白天鹅”都是动物，二者属于并列关系，排除；
“胚胎”是“婴儿”发育的一个阶段，但不是变态发育，排除；
“种子”是“花朵”发育的一个阶段，但不是变态发育，排除；
“青蛙”的发育过程大体经过四个时期：受精卵、“蝌蚪”、幼蛙、成蛙。“蝌蚪”是“青蛙”变态发育的一个阶段，二者属于对应关系，与题干逻辑关系一致，符合。
:::
