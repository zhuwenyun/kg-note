<p>第1题</p>
<p>下列关于党风建设的创新，按时间先后顺序排列正确的是：<br/>①以马克思列宁主义的理论思想武装起来的中国共产党，在中国人民中产生了新的工作作风，这主要的就是理论和实践相结合的作风，和人民群众紧密地联系在一起的作风以及自我批评的作风<br/>②工作作风上的问题绝对不是小事，如果不坚决纠正不良风气，任其发展下去，就会像一座无形的墙把我们党和人民群众隔开，我们党就会失去根基、失去血脉、失去力量<br/>③务必使同志们继续地保持谦虚、谨慎、不骄、不躁的作风，务必使同志们继续地保持艰苦奋斗的作风<br/>④抓精神文明建设、抓党风、社会风气好转，必须狠狠地抓，一天不放松地抓，从具体事件抓起</p>
    
<p>A. ①③②④</p>
<p>B. ③④①②</p>
<p>C. ③①④②</p>
<p>D. ①③④②</p>

::: details
<p>第一步，本题考查党风建设内容的时间先后顺序。<br/>第二步，①项：党的三大作风建设出自1945年中共七大毛泽东的《论联合政府》的报告；②项：是2013年1月习近平总书记在中央纪委十八届二次全会上对干部作风的重要作用做的批示；③项：“两个务必”是毛泽东在1949年3月七届二中全会指出的内容；④项：是1986年邓小平同志在中共中央政治局常委会的讲话；正确的顺序为①③④②。<br/>因此，选择D选项。</p>
:::
    

::: tip
<p>①项：中共七大于1945年4月在延安杨家岭召开，毛泽东在会上作了《论联合政府》的政治报告，提出了“三大作风”建设，并将毛泽东思想写入党章，确立为党的指导思想。<br/>②项：2013年1月21日至22日，中国共产党第十八届中央纪律检查委员会第二次全体会议在北京召开，习总书记在会上谈到干部作风问题时提出：“工作作风上的问题绝对不是小事……我们党就会失去根基、失去血脉、失去力量”。<br/>③项：七届二中全会于1949年3月在河北西柏坡召开，首次提出加强党的建设，提出“两个务必”思想内容，全会着重讨论了党的工作重心由农村转移到城市的问题，并确定了建国之后的两大任务：使中国稳步地由农业国转变为工业国，由新民主主义国家转变为社会主义国家。<br/>④项：1986年1月17日，邓小平在中共中央政治局常委会上讲话时说：“抓精神文明建设，……一天不放松地抓，从具体事件抓起。”</p>
:::
    


<p>第2题</p>
<p>关于宇航员在太空的生活，下列说法不正确的是：</p>
    
<p>A. 宇航员可使用特定的加热器对食品进行加热</p>
<p>B. 宇航员从太空返回地面后，失重状态消失，质量会有所增加</p>
<p>C. 宇航员应睡在固定的睡袋中，以免被气流推动误碰仪器设备开关</p>
<p>D. 在同一航空器中的宇航员可以直接交谈，无需借助无线电通讯设备</p>

::: details
<p>第一步，本题考查太空知识并选错误项。<br/>第二步，质量是物体本身的属性，它不随物体的位置、形状、温度及状态的改变而改变。B项中从太空回到地面，人的质量不会有所改变。<br/>因此，选择B选项。</p>
:::
    

::: tip
<p>A项：宇航员在太空舱里一般使用微波加热器来烘烤食物，这种微波加热器与地面上使用的加热器有所不同，它上面有一些特制的凹进去的小格，为了防止加热时食物飘浮起来，需要加热的食物都必须固定在这些小格内，插上电源后对食物进行加热。A项正确。<br/>C项：因为在太空之中，宇航员会处在失重状态。所以他们睡觉时必须睡在固定的睡袋里，睡袋一般固定在飞船的内舱壁上，为了防止无意中触及开关，他们睡觉时还必须把双手束在胸前。C项正确。<br/>D项：声音的传播需要介质，太空舱内有气体，同一航空器中的宇航员可以面对面直接说话交流。D项正确。</p>
:::
    


<p>第3题</p>
<p>下列关于国家主权及国防地理的表述，不正确的是：</p>
    
<p>A. 主权是联合国赋予国家的最基本的权利</p>
<p>B. 一国的领海和领空都是其领土的组成部分</p>
<p>C. 我国南海四大群岛是东沙、西沙、中沙和南沙群岛</p>
<p>D. 我国与越南、缅甸、吉尔吉斯斯坦等十几个国家接壤</p>

::: details
<p>第一步，本题考查国防地理并选错误项。<br/>第二步，对比选项，A项中主权是国家的最重要属性，是国家在国际法上所固有的独立处理对内对外事务的权力。它是国家固有的，并非是由联合国所赋予的。<br/>因此，选择A选项。</p>
:::
    

::: tip
<p>B项：国家领土是指在一国主权管辖下的区域，包括一国的陆地、河流、湖泊、内海、领海以及它们的底床、底土和上空（领空）。<br/>C项：南海诸岛为南海中中国许多岛礁的总称，属海南省和广东省，包括广泛分布的200多个岛、礁、沙滩。按其分布位置，分为4大群岛：东沙群岛、西沙群岛、中沙群岛、南沙群岛。<br/>D项：我国共有14个陆上邻国，分别是朝鲜、俄罗斯、蒙古、哈萨克斯坦、吉尔吉斯斯坦、塔吉克斯坦、阿富汗、巴基斯坦、印度、不丹、尼泊尔、缅甸、老挝和越南。<br/>解题技巧：本题的D选项可以使用速记口诀进行判断。口诀为：月娥姑娘很腼腆（越南、俄罗斯、缅甸），蒙着布单披三毯（蒙古、不丹、塔吉克斯坦、哈萨克斯坦、吉尔吉斯斯坦），度过泥巴去朝鲜（印度、老挝、尼泊尔、巴基斯坦、朝鲜），终于到了阿富汗（阿富汗）。</p>
:::
    


<p>第4题</p>
<p>下列情形可能发生的是：</p>
    
<p>A. 南北朝贵族妇女去佛寺礼佛</p>
<p>B. 半坡原始居民种植玉米</p>
<p>C. 周武王穿着铁制铠甲伐纣</p>
<p>D. 秦朝儿童春天放纸风筝</p>

::: details
<p>第一步，题干考查中国古代史。<br/>第二步，佛教是在西汉张骞通西域之后由印度传入我国的，南北朝是佛教兴盛时期（“南朝四百八十寺”），所以南北朝贵族妇女去佛寺礼佛是可能出现的。<br/>因此，选择A选项。</p>
:::
    

::: tip
<p>B项：玉米是重要的粮食作物和饲料作物，也是全世界总产量最高的农作物。它原产于拉丁美洲，在欧洲国家开辟新航路之后，由传教士引入到我国，而半坡原始居民还处于原始社会，所以B项不可能发生。<br/>C项：周武王是西周的开国之君，商周时期是青铜时代。而铁器产生于春秋时期，直到战国时期才得到广泛应用，所以C项不可能发生。<br/>D项：纸是我国古代的主要书写材料，西汉时期出现了造纸术，到东汉由蔡伦对造纸术进行了改进。而秦朝早于汉朝，当时还没有出现纸，所以D项不可能发生。</p>
:::

<p>A. ⑤④①③②</p>
<p>B. ⑤①③②④</p>
<p>C. ⑤③①②④</p>
<p>D. ③①⑤④②</p>

<p>第6题</p>
<p>《人民日报》评论指出：“一个人挥舞胳膊的自由止于别人鼻子的地方。”下列可以代替该评论的名言是：</p>
    
<p>A. 哪里没有法律，哪里就没有自由</p>
<p>B. 法典就是人民自由的圣经</p>
<p>C. 法律是自由的保姆</p>
<p>D. 自由只服从法律</p>

::: details
<p>第一步，本题考查法律与自由的关系。<br/>第二步，人民日报引用“一个人挥舞胳膊的自由止于别人鼻子的地方”意在说明，公民虽然可以自由行使权利，但必须在法律规定的范围之内行使权力，超出了法律规定的范围就没有自由可言。洛克曾说：“哪里没有法律，哪里就没有自由。”强调公民要依法行使权利和自由，和题目中一个人虽然可以随意挥舞自己的胳膊，但是也不能超过一定的界限相吻合。A项表述与题干表述一致。<br/>因此，选择A选项。</p>
:::
    

::: tip
<p>B项：马克思说：“法典就是人民自由的圣经。”说的意思是法律要保障人民的利益和自由，但题目的侧重点不在保障自由上，故不当选。<br/>C项：柏拉图说：“法律是自由的保姆。”跟B项类似，侧重点在保障自由上，故不当选。<br/>D项：伏尔泰说：“自由只服从法律。”确实自由的行使权利也是有边界的，但是“只”字题目没有体现出来，故不当选。</p>
:::
    


<p>第7题</p>
<p>下列法律谚语与其蕴含的法学理念对应正确的是：</p>
    
<p>A. 法无明文授权不得为：人的权利根源于法条</p>
<p>B. 在法律面前人人平等：人的自由不能被剥夺</p>
<p>C. 迟到的正义不是正义：效率是法的价值目标</p>
<p>D. 民若不告则官必不究：诉权只能由个人行使</p>

::: details
<p>第一步，本题考查法的价值目标。<br/>第二步，“迟到的正义不是正义”是英国的一句法律谚语，它强调了对司法程序正义的重要性。这句话的意思是说，即使司法裁判的结果是公正的，如果过迟做出裁判，或者过迟告知当事人，程序上的不公正将使裁判成为非正义的。“效率是法的价值目标”强调的正是效率对法律具有重要的意义。二者意思相吻合。<br/>因此，选择C选项。</p>
:::
    

::: tip
<p>A项：“法无明文授权不得为”指行政主体在行政过程之中应依法办事。“人的权利根源于法条”指人的权利由法律规定，我们行使权力的前提是遵纪守法。二者不对应。<br/>B项：“在法律面前人人平等”是指人人平等地享有法律规定的权利和承担法律义务，不允许任何人享有超越法律的特权。“人的自由不能被剥夺”本身说法错误，人在触犯法律的情况下是可以被剥夺自由的。二者不对应。<br/>D项：“民若不告则官必不究”主要是针对自诉案件而言。“诉权只能由个人行使”本身说法错误，在公诉案件中，诉权由检察院行使。二者不对应。</p>
:::
    


<p>第8题</p>
<p>我国古代文人在诗词中常运用典故表达自己的思想感受，下列作品中没有使用典故的是：</p>
    
<p>A. 桃花潭水深千尺，不及汪伦送我情</p>
<p>B. 蓬山此去无多路，青鸟殷勤为探看</p>
<p>C. 为报倾城随太守，亲射虎，看孙郎</p>
<p>D. 东篱把酒黄昏后，有暗香盈袖</p>

::: details
<p>第一步，本题考查诗词典故并选错误项。<br/>第二步，A项出自唐代诗人李白的《赠汪伦》，是李白于安徽泾县游历桃花潭时写给当地好友汪伦的一首留别诗。这句诗是说深达千尺的桃花潭水也比不上汪伦送别李白的深情厚谊，它运用了夸张的修辞方法，但并未使用典故。<br/>因此，选择A选项。</p>
:::
    

::: tip
<p>B项：出自唐代李商隐的《无题》。“蓬山”“青鸟”典故出自《山海经》。西王母居于蓬莱仙山之上，青鸟是为她取食传信的使者。<br/>C项：出自北宋苏轼的《江城子·密州出猎》。“亲射虎，看孙郎”典故出自《三国志》。“孙郎”指孙权，一次出巡中，他的坐骑为猛虎所伤，他仍镇定地在马前击毙猛虎。<br/>D项：出自北宋李清照的《醉花阴·薄雾浓云愁永昼》。“东篱”为菊圃的代称，典故出自陶渊明《饮酒》：“采菊东篱下，悠然见南山”；“有暗香盈袖”化用了《古诗十九首·庭中有奇树》：“馨香盈怀袖，路远莫致之”。</p>
:::
    


<p>第9题</p>
<p>俗话说“绣花要得手绵巧，打铁还须自身硬”，下列与该俗语哲学道理相同的是：</p>
    
<p>A. 身正不怕影子斜</p>
<p>B. 师傅领进门，修行靠个人</p>
<p>C. 水大漫不过船</p>
<p>D. 有理不在声高</p>

::: details
<p>第一步，本题考查哲学的内外因辩证关系原理。<br/>第二步，“绣花要得手绵巧，打铁还须自身硬”意思是说要绣出漂亮的花朵，手必须要灵活灵巧；要打出坚固的铁器，铁匠必须自身具备很强的能力，它们都是在强调内因的重要性。B项“师傅领进门，修行靠个人”是说虽然我们的成功需要有老师的指导，但老师的影响属于外部因素。而自身的努力学习才是取得成功的最重要的因素，它也强调了内因的重要性，与题干相对应。<br/>因此，选择B选项。</p>
:::
    

::: tip
<p>A项：“身正不怕影子斜”通常用来比喻只要为人做事走得正、行得端，就没有什么可害怕的。“影子”是现象，“身正”是本质，所以这体现了现象与本质的辩证关系原理。<br/>C项：“水大漫不过船”是说船只不管行驶在多深的水中，不论水涨得多高，总是不可能漫过船体。这体现的是我们生活的这个物质世界是客观的、有规律的，我们要尊重客观规律。<br/>D项：“有理不在声高”意指在争论中，关键看说得在不在理，而不在于嗓门大小。这体现的是真理的客观性原理。</p>
:::
    


<p>第10题</p>
<p>下列诗句按其所描写节日的先后顺序，排列正确的是：<br/>①桃符呵笔写，椒酒过花斟<br/>②九日黄花酒，登高会昔闻<br/>③粽包分两髻，艾束著危冠<br/>④马上逢寒食，途中属暮春</p>
    
<p>A. ④②③①</p>
<p>B. ①④③②</p>
<p>C. ②④③①</p>
<p>D. ③①②④</p>

::: details
<p>第一步，题干考查传统节日并按先后排序。<br/>第二步，①描写春节；②描写重阳节；③描写端午节；④描写寒食节。故正确排序是①④③②。<br/>因此，选择B选项。</p>
:::
    

::: tip
<p>①“桃符”是春联的别称，诗句描写春节，时间为农历正月初一。②“黄花酒”为菊花酒，饮菊花酒和登高为重阳节活动，时间为农历九月初九。③“粽包”和“艾束”分别指代食粽子、佩艾叶。吃粽子、饮雄黄酒、佩艾叶都是端午节的习俗，时间为农历五月初五。④“寒食”为古代节日，在夏历冬至后105日，在清明节的前一天，起源于晋文公为纪念介子推，禁烟火、吃冷食。另外，从“暮春”一词也可看出其时间。</p>
:::
    


<p>第11题</p>
<p>下列划线词语在古代和现代含义相同的是：</p>
    
<p>A. <u></u>阡陌<u>交通</u>，鸡犬相闻</p>
<p>B. <u>烈士</u>暮年，壮心不已</p>
<p>C. 今天下三分，益州疲弊，此诚<u>危急</u>存亡之秋也</p>
<p>D. 若舍郑以为东道主，<u>行李</u>之往来，共其乏困，君亦无所害</p>

::: details
<p>第一步，题干考查古代汉语的内涵。<br/>第二步，C项出自三国时期蜀国丞相诸葛亮所写的《前出师表》，文中“危急”的意思是指情况危险紧急，现代汉语中“危急”是指危险而急迫，所以它与现代含义相同，当选。<br/>因此，选择C选项。</p>
:::
    

::: tip

<p>下列关于自然现象的说法错误的是：</p>
    
<p>A. 霾通常呈黄色或橙灰色，吸入后有害</p>
<p>B. 露水会对农作物的生长造成危害</p>
<p>C. 雾和云是由于温度下降造成的</p>
<p>D. 彩虹有时会同时出现两条</p>

:::

<p>D. 翻过山脉后气流温度会有所上升</p>

<p>第一步，本题考查地形雨并选错误项。<br/>第二步，该图为地形雨示意图，地形雨是湿润气流遇到山脉等高地阻挡时被迫抬升而气温降低形成的降水。形成降水的山坡正好是迎风的一面，称为迎风坡。而背风的一面，因为气流下沉，温度升高，不再形成降水，称为背风坡。地形雨如不与对流雨或台风雨结合，雨势一般不会很强，降水持续时间也不会长，故C项说法错误。<br/>因此，选择C选项。</p>

<p>第15题</p>
<p>下列说法错误的是：</p>
    
<p>A. 根据光的偏振原理可以制成看立体电影的立体眼镜</p>
<p>B. 打枪瞄准时闭上一只眼利用了光沿直线传播的原理</p>
<p>C. 光在光纤中传播利用的是光的全反射原理</p>
<p>D. 潜望镜利用了光的折射原理</p>

::: details
<p>第一步，本题考查光学知识并选错误项。<br/>第二步，潜望镜是指从海面下伸出海面或从低洼坑道伸出地面，用以窥探海面或地面上活动的装置。它常用于潜水艇、坑道和坦克内用以观察敌情，其构造与普通地上望远镜相同，另加两个反射镜使物光经两次反射而折向眼中（具体见下图）。所以D项中的潜望镜是利用光的反射原理来制作而非光的折射原理。<br/><img src="http://tiku.huatu.com/cdn/pandora/img/26969669-1c18-4b2f-ac68-a8f02c4dfe52..jpg?imageView2/0/w/490/format/jpg" width="489" height="331"><br/>因此，选择D选项。</p>

<p>B. 局限 排斥 针锋相对</p>

<p>D. 拘泥 否定 相去甚远</p>

::: details
<p>第一步，突破口在第二空。从“只是试图用实验结果来证明这些假说的合理性，而不是……”可以看出，第二空应与“证明”意思相反。A项“反驳”指提出反对的理由辩驳；反对；D项“否定”指拒绝承认。二者均符合语境。B项“排斥”指排除驳斥，与“证明”构不成反义，排除B项。C项“求证”指寻找证据或求得证实，与“证明”意思相近，不符合文意，排除C项。<br/>第二步，再看第一空。A项“受制”意为想要突破却受到外界条件的限制或束缚。D项“拘泥”指固执成见而不知变通，强调自身主观上不愿去突破。文中“许多科学家只是去证明……而不是去……”说明了这些科学家正是因为主观上认为这些“假说”是对的，所以不愿去否定，故应选择“拘泥”。答案锁定D选项。<br/>第三步，验证第三空。“相去甚远”指互相之间存在很大差异和距离，符合语境。<br/>因此，选择D选项。</p>

<p>第95题</p>
<p>红旗原则是指如果侵犯著作权的事实是显而易见的，网络服务商就不能装作看不见，或以不知道侵权的理由来推脱责任。<br/>根据上述定义，下列适用红旗原则的情形是：</p>

<p>D. 丙某通过第三方在有下载功能的互联网电视上获取未经授权的节目</p>

::: details
<p>第一步，看提问方式，本题属于选是题。<br/>第二步，找关键信息。<br/>①侵犯著作权的事实是显而易见的；<br/>②网络服务商就不能装作看不见，或以不知道侵权的理由来推脱责任。<br/>第三步，辨析选项。<br/>A项：“未经用户许可获取用户个人数据”不涉及著作权，不符合定义；<br/>B项：“经营盗版影视作品”说明侵犯了著作权，但甲某利用的是网站免费存储空间，网络中介商没有能力事先对内容进行审查，不体现“侵权事实是显而易见的”，不符合定义；<br/>C项：正在首播的电视剧被乙某上传，说明乙某“侵犯该影视作品著作权”，被网站编辑推荐到下载首页，说明网站知晓该事件的存在，体现了“侵权的事实显而易见”，符合定义；<br/>D项：“互联网电视”也可以通过电视服务商提供服务，无法确定是否属于网络服务商提供的服务，不符合定义。<br/>因此，选择C选项。</p>

<p>第一步，本题考查综合分析问题，且需选出正确的一项。<br/>第二步，A选项，读数比较。定位表2，2010年，新能源发电机组上网电量（492亿千瓦时）已经超过核电上网电量（308亿千瓦时），错误。<br/>B选项，年均增长率比较。2008年—2012年，水电上网电量和水电并网容量的年均增长率比较大小。根据末期量=初期量×（1+年均增长率）^n，由于年份差相同，比较年均增长率大小只需比较末期量/初期量的大小。水电上网电量为<img src="https://pandora-pro.oss-cn-beijing.aliyuncs.com/pandora/img/91dfd5f9242744629ac087c9be7f542b.png" width="140.02501" height="35.625" style="width:140.02501;height:35.625" data-latex="\frac{5518}{3662}\approx\frac{5518}{3660}\approx1.5^+">，并网容量为<img src="https://pandora-pro.oss-cn-beijing.aliyuncs.com/pandora/img/fb8da58cf46244329330915f693086fc.png" width="155.02501" height="35.625" style="width:155.02501;height:35.625" data-latex="\frac{16816}{12214}\approx\frac{16816}{12200}\approx1.4^-">，水电上网电量年均增长率超过同期水电并网容量，正确。<br/>C选项，现期倍数。定位表1，2012年新能源发电机组并网容量是2008年的<img src="https://pandora-pro.oss-cn-beijing.aliyuncs.com/pandora/img/3ca12aa9a81e456c92ad0e0d56143ea3.png" width="66.525" height="35.625" style="width:66.525;height:35.625" data-latex="\frac{6548}{803}\approx8">倍，即翻3番，错误。<br/>D选项，读数比较。定位表2，与上年相比，2010年核电上网电量（308亿千瓦时）＜2009年核电上网电量（355亿千瓦时），所以核电上网电量不是保持持续增长，错误。<br/>因此，选择B选项。</p>
